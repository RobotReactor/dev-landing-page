SlideQuest

Slide Quest is a single-player puzzle game designed with challenging sliding mechanics that require players to think strategically to complete each level.

The game uses an algorithm to determine optimal moves and create challenging puzzles. Players navigate a grid by sliding along ice tiles in specific directions, aiming to solve the puzzle with the fewest moves possible.

A key feature of Slide Quest is its Level Editor, which allows users to create and customize their own puzzles and save snippets to generate levels in future builds.

Although a work in progress, completion of game sprites and algorithm generated maps using user created chunks are next up in the pipeline.

Workout Tracker Website (In Progress)

Currently in development, this workout tracking website will provide users with a seamless way to manage their fitness routines and track their progress. Built with .NET ASP.NET and Blazor for the front end, the website will allow users to sign in and securely store their personal workout data in an SQL database. Key features include a calorie intake calculator, customizable exercise creation, workout split management, and tracking of sets and reps. The site will also feature an intuitive calendar for scheduling workouts, making it easier to organize and view fitness routines. The website will be optimized for mobile use, allowing users to track workouts on the go.

A video demo and GitHub link will be provided soon, with more updates to come as development continues.
